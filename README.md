# Mudstone + Fractal
![Mud](http://ournameismud.co.uk/css/images/maps-icon.png)

Frontend starter kit integrated with fractal
http://fractal.build/

###### Recommended requirements
- node >= 8.4.0
- gulp >= 3.91
- yarn >= 0.27.5
- webpack >= 3.0.0

###### Installation

`yarn`

###### Development mode

`yarn start`

###### Build component library
`yarn run build`

###### Build for craftcms
`yarn run build:cms`

###### Take regression test reference
`yarn run reference`

###### Regression test diff 
`yarn run diff`
